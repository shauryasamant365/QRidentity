Shaurya Samant

He's the Team Leader of QRidentity. He had initiated the project QRidentity and has written all the code behind this website.

We've seen our teachers working day and night brightning our future. And during this process, working with attendance is a headache. A proper record of Attendance is need for every meeting and maintaining that is a very difficult and time consuming work. So, our team came up with a idea of building a system to handle all this stuff. We call this QRidentity, in which attendance is marked by scanning QR code and is saved in a SQLite database.
